This MATLAB FileExchange entry contains files related to the "Importing and Visualizing Videos" video of the AUVSI Foundation: Computer Vision Training.

examples - Files used in the video
[~] playVideoFile.m : Example MATLAB code file to import and play a video file in MATLAB.
		      Additionally, shows a couple of methods to explore system objects.
[~] acquireImages.m : Example MATLAB code file to acquire 'N' # of frames from a camera.
[~] buoyRun.avi : Example video of a robot boat navigating a field of red and green buoys.

exercises - Files for exercise and practice tasks 
[~] auvsi_ipcv_exercises_01_note.pdf : Note introducing the exercise tasks and presenting respective solutions
[~] importVideoFile.m : Solution for "Video File Import" exercise. Shows how to import and view video from a file in 			                MATLAB.
[~] acquireVideoCamera.m : Solution for "Video Camera Acquisition" exercise. Shows how to acquire video from a camera in 			   MATLAB.
